Percentage Overlap

This function calculates the percentage of overlapping peaks in each file. By default, two peak ranges are considered as overlapping if:

  • The start or end position of a range is inside the other range
  • The overlap has a minimum distance of 1 base pair
  • The overlap can be of any type (i.e. start, end, within, equal are all accepted)
